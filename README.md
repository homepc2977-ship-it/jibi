# JIBI

JIBI is a mobile-first budgeting app that helps users know what money is safe to spend before the next payday.

## Core idea

**Know what you can safely spend before your next payday.**

## MVP Modules

- Dashboard with Safe-to-Spend number
- Budget setup
- Fixed charges and payment reminders
- Calendar view for paydays and bills
- Goals: house, car, savings, emergency fund, debt payoff
- Goal modes: Extreme, Normal, Comfortable
- Household/shared budget
- Group split expenses
- Reports and analytics

## UI Direction

- Mobile-first
- Soft mint/green branding
- Card-based dashboard
- Large money numbers
- Friendly onboarding
- Bottom navigation

## Status

Project initialized as a clean starter for the JIBI MVP.
